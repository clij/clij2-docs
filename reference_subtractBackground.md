## subtractBackground
<img src="images/mini_empty_logo.png"/><img src="images/mini_empty_logo.png"/><img src="images/mini_empty_logo.png"/><img src="images/mini_empty_logo.png"/>

Applies Gaussian blur to the input image and subtracts the result from the original image.

Deprecated: Use topHat() or differenceOfGaussian() instead.

### Usage in ImageJ macro
```
Ext.CLIJx_subtractBackground(Image input, Image destination, Number sigmaX, Number sigmaY);
```
